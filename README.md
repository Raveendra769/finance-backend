# Finance Dashboard Backend

## Features
- User Login & Register
- Role-Based Access (Admin, Analyst, Viewer)
- Financial Records CRUD
- Dashboard Summary (Income, Expense, Balance)

## Tech Stack
- Node.js
- Express
- MongoDB

## API
- POST /api/users/register
- POST /api/users/login
- POST /api/records
- GET /api/records
- GET /api/dashboard
